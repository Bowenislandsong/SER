# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- GitHub Pages documentation website
- Multiple installation methods (PyPI, Conda, Docker)
- Automated release workflows for PyPI and Docker
- Comprehensive release documentation (RELEASE.md)
- Docker support with Jupyter notebook
- Conda package configuration
- Badges and documentation links in README

### Changed
- Improved README with all installation methods
- Updated PyPI workflow to trigger on releases instead of pushes
- Enhanced pyproject.toml with additional project URLs

## [0.1.0] - 2024

### Added
- Initial release
- SVD Embedding Regression algorithm
- Distributed SVD algorithm
- Federated SVD algorithm
- Basic test suite
- Examples for all algorithms
- README with usage instructions
- GPL-3.0 license
